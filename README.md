Giới thiệu Dự án được thực hiện nhằm xây dựng một nền tảng trực tuyến giúp cộng đồng tìm kiếm việc làm.

Về mặt kỹ thuật, hệ thống được xây dựng dựa trên mô hình MVC (Model – View – Controller), giúp cấu trúc mã nguồn rõ ràng, dễ quản lý, thuận tiện cho việc bảo trì cũng như mở rộng thêm các chức năng trong tương lai mà không làm ảnh hưởng đến các thành phần hiện có.

Công nghệ sử dụng:

Ngôn ngữ lập trình: C# (C-Sharp)

Framework: ASP.NET MVC

ORM: Entity Framework

Cơ sở dữ liệu: SQL Server

Giao diện: HTML, CSS, JavaScript, Bootstrap

Các phân hệ chính của hệ thống:

Phân hệ khách vãng lai:
Cho phép người dùng truy cập trang chủ, tìm kiếm việc làm, xem thông tin chi tiết của từng công việc và đăng ký tài khoản mới.

Phân hệ người tìm việc:
Hỗ trợ đăng nhập/đăng xuất, nộp hồ sơ ứng tuyển, cập nhật và quản lý thông tin cá nhân.

Phân hệ nhà tuyển dụng:
Cho phép đăng nhập/đăng xuất, đăng tin tuyển dụng, chỉnh sửa và quản lý các tin đã đăng, theo dõi danh sách ứng viên và cập nhật trạng thái tuyển dụng.

Phân hệ quản trị viên:
Đảm bảo đăng nhập an toàn, cung cấp các chức năng thống kê tổng quan hệ thống, quản lý người dùng và kiểm duyệt, quản lý các tin tuyển dụng.
